What's Included

On-Device Agent

Lightweight Rust HTTP server (~0.8 MB RSS) replaces ZTE's 225 MB daemon stack. Signal monitoring, band locking, config backup, and 40+ REST endpoints.

Mobile Companion Apps

Native iOS (SwiftUI) and Android (Compose) apps. Dashboard, signal monitor, SMS, band lock, device management — all over WiFi.

Web Bootstrap

Deploy the agent entirely from your browser. WebUSB-powered — no command line needed.

Deploy Agent

Connect to Your Router

Connect to your U60 Pro's WiFi network (or use the default Ethernet connection).

Enter Credentials

Router admin password to enable ADB, and a password for the zte-agent API.

Enabling ADB…

Authenticating with your router and switching USB mode. This takes a few seconds.

Connect USB Cable

ADB is enabled. Now connect your U60 Pro to this computer with a USB-C cable, then click below.

A browser prompt will ask you to select the USB device.
WebUSB is not supported in this browser. Please use Chrome or Edge on desktop.

Deploying Agent…

Downloading, pushing, and starting zte-agent on your router.

Setup Complete!

The zte-agent is running on your router and will auto-start on reboot.

Next steps:

1. Download the companion app for iOS or Android.

2. Connect to your router's WiFi and open the app.

3. Log in with the agent password you set above.

Agent API endpoint:

http://192.168.0.1:9090
